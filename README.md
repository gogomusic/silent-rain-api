## 描述

这是网站“静夜聆雨”的后端仓库项目，使用 `NestJS` 编写

## 项目设置

```bash
$ pnpm install
```

## 编译和运行项目

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## 运行测试

> [!warning]
>
> 单元测试、e2e测试还没学会，项目中暂未添加测试代码
